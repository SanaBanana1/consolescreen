Console Screen

To set visual studio code up you have to first create a blank folder on your computer then open it in the studio. Once you have done that you then create a files named index.html, test.html and style,css to do basic things.

The importance of console screens is for viewing outputs before actually putting it out on the HTML page. To open the console screen you must tap you computer mouse with 2 fingers and press inspect. After doing that you have to click on the console option.
